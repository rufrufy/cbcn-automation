Langkah berikutnya:
  1. Download .5sim_key dan .env ke folder repo (root).
  2. Install deps: pip install requests beautifulsoup4
  3. Pastikan 9router jalan di port 20128 (DB: ~/.9router/db/data.sqlite).
  4. Copy Run Command di atas, paste di terminal di folder repo, tekan Enter.
  5. Hasil key ada di API_KEYS.txt & otomatis inject ke 9router DB.
